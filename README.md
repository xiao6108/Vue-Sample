## Vue-cli, pug, sass, Webpack, bootstrap4
VueCli Webpack打包網頁，pug、sass、scss編譯
安裝bootstrap4
Navbar可切換Vue-router頁面

## Build Setup
``` bash
# install dependencies
npm install

# open in browser localhost:8080
npm run dev

# build for production with minification
npm run build

```

## Vue Cli 環境需求
